News，一款新闻阅读App,本app由小帅和图图共同开发，基于 Material Design  + RxJava + Retrofit + Glide，基本涵盖了当前 Android 端开发常用的主流框架，基于此框架可以快速开发一个app。
目前模块包括：

新闻：头条、科技、财经、NBA等类型
美图：美图壁纸
笑话:经典笑话
视频：包括热点、搞笑、娱乐、精品视频


RxJava:包括Rx处理服务器请求、缓存、线程调度的完美封装
